Q1. What is difference between Map, Filter , Find and ForEach?

Map, filter, find, and forEach are array methods that iterate over arrays. Give some details on its every topics:

  1. 1.Map: Map creates a new array by given array. It takes a function as an argument and returns a new array with the same length as the original array, map are creating element based on given array.
  2. 2.Filter: Filter creates a new array by filtering elements from the given array. It takes a function as an argument and returns a new array of elements that are fulfill condition.
  3. 3.Find: Find returns the first element in the array that fulfill the condition.If no elements fill the condition, it returns undefined.
  4. 4.ForEach: ForEach going over the elements in the array and performs a function on each element. forEach does not create a new array or return a value.

Q2. What is difference between Local Storage and Session Storage?

localStorage and sessionStorage are two browser storage that store data in the browser. The main difference between them is the time of the stored data; data in localStorage are stored if tab or browser closed or pc shutdown but data in sessionStorage is cleared when the browser is closed or the tab is closed or px shutdown.
